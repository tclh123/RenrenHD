Readme for Microsoft Visual Studio International Pack 1.0

This readme provides information about what's new in Visual Studio International Pack 1.0, which is our first release to support the international features for .NET developers, and known issues in Visual Studio International Pack 1.0.

What's in Visual Studio International Pack 1.0

  • East Asia Numeric Formatting Library

    • East Asia Numeric Formatting Library provides class library that helps developers formatting numeric data into Traditional Chinese, Simplified Chinese, Japanese and Korean capital style string.

  • Japanese Kana Conversion Library

    • Japanese Kana Conversion Library provides class library that helps developer to convert Japanese characters into Kana symbols.

  • Japanese Text Alignment Library

    • Japanese Text Alignment Library provides class library which enables applications to render strings in Japanese specific alignment style.

  • Japanese Yomi Auto-Completion Library

    • Japanese Yomi Auto-Completion Library provides class library and TextBox control sample that support Japanese IME-aware auto-complete. This library is able to work only if Japanese IME is installed and activated.

  • Korean Auto Complete TextBox Control

    • Korean Auto Complete TextBox Control provides TextBox control that supports auto-complete and Intellisense feature with Korean input.

  • Simplified Chinese Pin-Yin Conversion Library

    • Simplified Chinese Pin-Yin Conversion Library provides several APIs that are designed to retrieve characters' properties such polyphone, homophone, Pinyin or stroke number for commonly used Simplified Chinese Characters.

  • Traditional Chinese to Simplified Chinese Conversion Library and Add-In Tool

    • Traditional to Simplified Chinese Conversion Library and Add-In Tool provides class library that helps developers to convert text in Traditional Chinese into Simplified Chinese or vice versa. The conversion leverage the engine built in Office 2007 that converts Traditional Chinese and Simplified Chinese in word-to-word mapping. This component also includes an add-in tool in VSIDE that can convert the Simplified Chinese resource file into Traditional Chinese.

System Requirement of Visual Studio International Pack 1.0

  • Microsoft Visual Studio 2005 or Microsoft Visual Studio 2008

  • Microsoft Office 2007 – This is optional for Traditional to Simplified Chinese Conversion Library and Add-In Tool. For better conversion result of Traditional/Simplified Chinese Conversion, user need to install Microsoft Office 2007 in the same machine to leverage the conversion engine inside.
Known Issues in the Visual Studio International Pack 1.0

  • Traditional to Simplified Chinese Conversion Library and Add-In Tool : When user converts the resource file with .zh-TW and .zh-CN culture extensions, the TCSC Conversion Add-In tool will generate the corresponding resource files with .zh-CHT and .zh-CHS culture extension. This is by design, since zh-TW and zh-CN are specific culture for zh-CHT and zh-CHS. Under the parent neutral culture, there is more than one specific culture. The add-in tool can not decide which specific culture should be converted to another. In our Add-In tool, we will convert the resource files with specific culture extension to neutral culture of the other Chinese language.
